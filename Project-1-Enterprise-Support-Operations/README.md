# Enterprise Support Operations Dashboard

## Objective
Built an operational analytics dashboard to monitor enterprise support performance across payroll, HCM, reporting, and tax systems.

## Tools Used
- Power BI
- Excel
- Operational Analytics
- Data Visualization

## Key Metrics
- Total Tickets
- SLA Performance
- Escalation Rate
- Resolution Time by Severity
- Business Impact Distribution
- Root Cause Analysis

## Business Insights
- Payroll and HCM modules generated the highest ticket volume
- Critical issues showed significantly higher resolution time
- Integration and data sync failures were major root causes
- SLA compliance remained above target thresholds
